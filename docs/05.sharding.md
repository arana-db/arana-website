# Sharding

## Config

```yaml
sharding_rule:
  tables:
    - name: employees.student
      allow_full_scan: true
      sequence:
        type: snowflake
        option:
      db_rules:
        - column: uid
          type: scriptExpr
          expr: parseInt($value % 32 / 8)
      tbl_rules:
        - column: uid
          type: scriptExpr
          expr: $value % 32
          step: 32
      topology:
        db_pattern: employees_${0000..0003}
        tbl_pattern: student_${0000..0031}
      attributes:
        sqlMaxLimit: -1
```
